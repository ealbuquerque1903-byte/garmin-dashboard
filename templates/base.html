🔑 Configurar acesso

Para atualizar os dados, é necessário um GitHub Personal Access Token com permissão workflow. Ele fica salvo só no seu aparelho.

Acesse github.com → Settings → Developer settings → Personal access tokens → Fine-grained e crie um token com permissão Actions: Read and write neste repositório.
